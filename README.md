# Comandos básicos de Git

## git --version
**Definición:** Muestra la versión de Git instalada en el sistema.  
**Ejemplo:**
git --version

---

## git init
**Definición:** Inicializa un repositorio Git en la carpeta actual.  
**Ejemplo:**
git init

---

## git add
**Definición:** Agrega archivos al área de preparación (staging).  
**Ejemplo:**
git add .

---

## git commit
**Definición:** Guarda los cambios con un mensaje descriptivo.  
**Ejemplo:**
git commit -m "Primer commit"

---

## git status
**Definición:** Muestra el estado actual del repositorio.  
**Ejemplo:**
git status

---

## Flujo básico de trabajo
git add .
git commit -m "mensaje"
git status
